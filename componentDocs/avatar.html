    # 修改默认图标

    <template>
  <hub-avatar
    shape="circle"
    icon="el-icon-user"
  />
</template>
    Expand Copy Copy

    # 属性

    参数 说明 类型 可选值 默认值
    icon 设置头像的图标类型 String - -
    shape 指定头像的形状 String circle/square circle
    size 设置头像的大小 String - 100
    src 图片类头像的资源地址 String - -
